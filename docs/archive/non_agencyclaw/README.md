# Non-AgencyClaw Docs Archive

This folder contains documentation that is not part of current AgencyClaw work.

Purpose:
- preserve historical PRDs/plans/specs without deleting context,
- keep root `docs/` focused on active AgencyClaw documentation,
- provide reversible archival (paths are still in git history).

Typical contents:
- legacy PRDs and implementation plans for shipped tools,
- old review summaries and deprecation plans,
- historical composer/scribe/debrief/command-center design notes.

If any non-AgencyClaw initiative is reactivated, move only the relevant docs back to an active location and update links accordingly.

For database truth, do not rely on archive schema snapshots; use `docs/db/schema_master.md`.
